### Extensions and Resources

The resources below are the ones I listed in my presentation. I also included this page to show that if you need to write a lot of text, you can use `includeMarkdown(path_to_doc)` to insert a markdown file in the UI. 

#### Tables
  - [`DT`](https://rstudio.github.io/DT/)
  - [`reactables`](https://glin.github.io/reactable/index.html)

#### Input Controls
  - [`shinywidgets`](https://shinyapps.dreamrs.fr/shinyWidgets/)

#### Themes and Layouts
  - [`bslib`](https://rstudio.github.io/bslib/)
  - [`shinydashboard`](https://rstudio.github.io/shinydashboard/)

#### Deploying
  - [shinyapps.io](https://www.shinyapps.io/)

#### Guides and Tutorials
  - [*Mastering Shiny* by Hadley Wickham](https://mastering-shiny.org/)
  - [A comprehensive collection of extensions for Shiny](https://github.com/nanxstats/awesome-shiny-extensions)
